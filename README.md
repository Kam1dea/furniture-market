Разрабатываемый backend для маркетплейса с поддержкой товаров и отзывов. Реализованы сущности Product, Review, JWT-аутентификация, глобальный обработчик исключений и репозитории на EF Core. Проект строится по принципам Clean Architecture и использованием DI. Планируется добавление профилей продавцов и загрузки изображений. Технологии: ASP.NET Core, PostgreSQL.
